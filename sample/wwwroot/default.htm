Zyin.IntentBot sample bot is ready!


You can test your bot in the Bot Framework Emulator by connecting to http://localhost:3978/api/messages.

If you want to test OAuth scenario, please set below settings in appsettings.json. You can use dotnet user secrets if you want to be more secure.
  • MicrosoftAppId - your bot id
  • MicrosoftAppPassword - your bot password
  • AADAppId - your Azure AD app id. Note this is different from your bot id.
  • AADAppPassword - your Azure AD app password.
  • OAuthConnectionName - The "OAuth Connection Setting" you configured in your Azure bot registration. Please set scope to "User.Read".
